## Installation

Clone the repository:

```bash
git clone https://github.com/dainyjose/my-portfolio.git
cd my-portfolio
```

Install dependencies:

```bash
  bun install
```

---

## Usage

Run the development server:

```bash
  bun run dev
```

Open in browser:

```bash
http://localhost:5173/
```

Build for production:

```bash
bun run build
```

Preview production build locally:

```bash
bun run preview
```

---

## Deployment

To deploy on GitHub Pages:

```bash
bun run deploy
```
